# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via one of the following channels:

- **GitHub Security Advisory**: [Report a vulnerability](https://github.com/Gen-x-academy/chainVerse-onchain/security/advisories/new)
- **Email**: security@gen-x-academy.com

Include as much detail as possible: affected contract(s), steps to reproduce, and potential impact.

## Expected Response Time

| Stage | Time |
|---|---|
| Initial acknowledgement | 48 hours |
| Status update | 5 business days |
| Fix or mitigation | 30 days (critical), 90 days (others) |

## Scope

The following smart contracts are in scope:

- `contracts/escrow`
- `contracts/escrow-vault`
- `contracts/certificates`
- `contracts/chainverse-core`
- `contracts/course_registry`
- `contracts/payout-automation`
- `contracts/reward`
- `contracts/shared`
- `contracts/token`

## Out of Scope

- Frontend code under `frontend/`
- Documentation files
- Test files
- Third-party dependencies
- Issues already publicly known or previously reported
